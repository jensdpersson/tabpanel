This is a demo and documentation page for the tabpanel utility. It consists of a CSS and a javascript function that agree on a set of css-classes with different semantics.
Usage CSS source Javascript source
There are many ways one could create tab layouts in css, each with pros and cons. This utility takes the approach to use one div for the tab-panel, annotated with the tabs css class. This then contains one div for the clickable tab titles. This div has the class tab-titles and contains a elements with class tab-title or tab-title-selected. At most one should have the latter. After the tab-titles div there is a series of divs with classes tab-content and tab-content-selected. The selected tab content should be at the same index among tab-content elements within this tab-panel as its tab-title is within the title div.
